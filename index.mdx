## What is Flamegraph?

Flamegraph is a VS Code extension that integrates two best-in-class Python profilers — [py-spy](https://github.com/benfred/py-spy) and [memray](https://github.com/bloomberg/memray) — directly into your editor. Profile Python scripts and Jupyter notebooks without modifying your code, then explore results in an interactive flamegraph panel with line-level annotations inside your source files.

## How it works

The extension profiles your code using py-spy or memray and renders results in two linked views:

1. **Flamegraph panel** — an interactive visualization of the full call stack, with zoom, module filtering, and code display toggle.
2. **Inline annotations** — per-line timing or memory decorations in your editor, synchronized with the flamegraph selection.

When you click a node in the flamegraph, the inline annotations update to show only the timing data for that portion of the call stack. Use `Cmd+Click` (Mac) or `Ctrl+Click` (Windows/Linux) on any flamegraph frame to open the corresponding source file.
